This is a simple password manager that has three functions.

The first function is to generate a complex password that has between 10 to 18 characters and then copies the generated password into the clipboard.

The second function is to save the provided information (website, Username/Email, Password) into a JSON file that will be created in the root folder.

The third function is to search for the Username/Email and Password combination for the given website. Where the combination will be ready to be copied in the relevant fields.
